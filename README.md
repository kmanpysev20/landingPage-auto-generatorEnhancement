# 랜딩페이지 생성기 V3 · 직접 편집형 (jQuery)

## 실행

- `index.html`을 Chrome 또는 Edge에서 엽니다.
- 파일 하나로 확인하려면 `landing-builder-direct-edit-single.html`을 엽니다.

## 새로 추가된 직접 편집 기능

### 미리보기에서 텍스트·버튼 이동

1. 중앙 모바일 미리보기에서 제목, 부제목, 상단 문구, 로고 또는 버튼을 클릭합니다.
2. 마우스로 잡아 원하는 위치까지 이동합니다.
3. 드래그 중 Shift를 누르면 처음 움직인 방향에 따라 가로 또는 세로축으로 고정됩니다.
4. 드래그 중 Ctrl(Mac은 Cmd)을 누르면 위치가 10px 단위로 맞춰집니다. Shift와 함께 사용할 수도 있습니다.
5. Ctrl(Mac은 Cmd)을 누른 채 다른 텍스트나 버튼을 클릭하면 여러 요소를 선택할 수 있습니다.
6. 요소 하나 또는 여러 요소를 섹션의 왼쪽·가운데·오른쪽에 배치할 수 있으며, 여러 요소는 간격을 유지한 하나의 묶음으로 이동합니다.
7. 선택한 요소를 `선택 요소 맨 위로` 버튼으로 다른 요소보다 위에 표시할 수 있습니다.
8. 오른쪽 `미리보기 요소 위치`에서 X·Y 좌표를 직접 입력할 수도 있습니다.
9. 방향키로 1px, Shift + 방향키로 10px씩 미세 조정할 수 있습니다.
10. `선택 요소 원위치` 또는 `섹션 전체 원위치`로 초기화할 수 있습니다.

### 미리보기에서 섹션 순서 변경

1. 이동할 섹션 위에 마우스를 올립니다.
2. 섹션 상단에 나타나는 `섹션 이동` 손잡이를 잡습니다.
3. 위 또는 아래 섹션으로 드래그하면 즉시 순서가 변경됩니다.
4. 왼쪽 섹션 목록에도 변경된 순서가 동시에 반영됩니다.

## 전체 기능

- 뷰티·코스메틱 / 제약·인증 / 패션·쇼핑몰 템플릿
- 미리보기 화면에서 섹션 직접 순서 변경
- 미리보기 화면에서 텍스트와 버튼 자유 이동
- 좌측 목록에서 섹션 추가, 복제, 삭제, 표시·숨김, 순서 변경
- 섹션 종류를 히어로·섹션·푸터 세 가지로 구성
- 제목, 부제목, 버튼 문구, 버튼 링크 실시간 편집
- 모든 섹션에서 텍스트·버튼·이미지를 여러 개 추가·삭제 가능
- 마지막 텍스트·버튼을 삭제하면 빈 편집 슬롯을 하나 유지해 다시 입력할 수 있음
- 히어로·제품·안내 이미지 업로드
- 섹션 이미지와 히어로 배경 이미지의 밝기·어둡기 조절
- 선택한 섹션·텍스트·버튼별 배경색과 테두리 편집
- 텍스트·버튼별 기본 글꼴, 글자 크기·굵기·색상 편집
- 버튼별 모서리와 밝기·어둡기 편집
- 모바일·태블릿 미리보기
- 미리보기 가로 폭을 280px부터 1024px까지 직접 조절
- 확대·축소, 자동 저장, 실행 취소·다시 실행
- 전체 미리보기 및 HTML 다운로드

## 번역 API 설정

- 상단 `번역`을 누르고 원문 언어, 번역 언어와 범위를 선택한 뒤 `번역 적용`을 누릅니다.
- 번역은 기존 HiddenTag API `https://hiddentagiqr.com:8050/api/app/translation`을 사용하므로 클라이언트에서 Google API 키를 입력하지 않습니다.
- 번역 전 원문을 작업별로 보관하므로 `현재 작업 원문 복원`으로 되돌릴 수 있습니다.
- 같은 원문과 언어로 이미 번역한 결과는 작업 데이터에서 재사용하여 중복 요청을 줄입니다.
- 월 500,000자 보호 한도는 이 브라우저에서 성공한 요청만 집계합니다. 기존 API를 사용하는 다른 서비스의 사용량까지 합산하려면 서버에서 중앙 사용량을 제공해야 합니다.
